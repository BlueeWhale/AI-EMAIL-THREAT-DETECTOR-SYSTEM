AI Email Threat Detector System

📌 Overview

The AI Email Threat Detector System is a machine learning-based web application that detects and classifies emails as Spam, Phishing, or Safe (Legitimate). The system analyzes email content using Natural Language Processing (NLP) techniques and predicts whether an email contains malicious intent.

This project helps users stay protected from cyber threats like phishing attacks, spam messages, and fraudulent emails.

🚀 Features

✅ Spam Email Detection

✅ Phishing Email Identification

✅ Safe Email Classification

✅ Machine Learning Model (Scikit-learn)

✅ NLP-based Text Processing (TF-IDF)

✅ Simple and Responsive Frontend (HTML, CSS, JavaScript)

✅ Python Backend (Flask)

✅ Real-time Email Analysis

⚙️ How It Works

User enters email subject and content.

Text is preprocessed (cleaning, stopword removal, tokenization).

TF-IDF converts text into numerical features.

ML model predicts the category:

Spam

Phishing

Safe

Result is displayed instantly on the dashboard.
